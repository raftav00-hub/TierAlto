Add the following conversion optimization elements 
to this website. Each element should be:
- Fully responsive (mobile and desktop)
- Smooth animated transitions
- Easy to customize via variables at the 
  top of each component
- Non-intrusive to user experience
- Dismissable where appropriate
- Session aware (don't show again if dismissed)

Build each element as a separate reusable 
component so they can be turned on or off 
independently.

─────────────────────────────────────────
ELEMENT 1 — EXIT INTENT POPUP
─────────────────────────────────────────
Trigger when user moves mouse toward 
the top of the browser.

Include:
- Dark overlay background
- Centered modal card
- Headline (customizable)
- Subheadline (customizable)
- Primary CTA button (customizable)
- Secondary dismiss text link
- Close X button in top right corner
- Do not show again once dismissed 
  in the same session
- Desktop only — do not trigger on mobile
- Fade in animation

─────────────────────────────────────────
ELEMENT 2 — STICKY HEADER
─────────────────────────────────────────
Make the main navigation header fixed 
to the top of the screen on scroll.

Include:
- Transparent background at top of page
- Solid white background with subtle 
  drop shadow when user scrolls down
- Logo on the left
- Navigation links in the center
- Primary CTA button on the right
- Smooth background transition on scroll
- Fully responsive with hamburger 
  menu on mobile

─────────────────────────────────────────
ELEMENT 3 — HELLO BAR / ANNOUNCEMENT BAR
─────────────────────────────────────────
A thin bar pinned to the very top of 
the page above the navigation.

Include:
- Customizable background color
- Short announcement text (customizable)
- Optional emoji support
- Optional link within the text
- Close X button on the right
- Remembers dismissed state in 
  localStorage
- Full width across all screen sizes

─────────────────────────────────────────
ELEMENT 4 — STICKY BOTTOM CTA BAR
─────────────────────────────────────────
A bar fixed to the bottom of the screen.

Include:
- Appears after user scrolls 300px
- Short headline on the left
- CTA button on the right
- Close/dismiss X button
- Slides up from bottom on appearance
- Show on mobile and desktop
- Customizable background color
- Customizable button color and text

─────────────────────────────────────────
ELEMENT 5 — TIMED POPUP
─────────────────────────────────────────
A popup that appears after the user 
has been on the page for a set time.

Include:
- Configurable delay timer in seconds

- Centered modal card
- Headline (customizable)
- Subheadline (customizable)

- Close X button
- Do not show again if dismissed

- Fade in animation

scrolls to a configurable percentage

- Fade in animation

─────────────────────────────────────────
IMPLEMENTATION NOTES
─────────────────────────────────────────
- Build each element as its own 
  component file
- Import and activate them all in 
  the main layout file so they appear 
  on every page
- Use CSS transitions and animations 
  — no heavy animation libraries
- All z-index values should be 
  coordinated so elements don't 
  overlap incorrectly
- Test all elements on Chrome, 
  Safari, Firefox and Edge
- Test all elements on iPhone and 
  Android screen sizes
- No element should block the main 
  CTA or form on any screen size
